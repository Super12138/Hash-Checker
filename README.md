# Hash-Checker
A free, open source, easy-to-use hash checker.<br/>
一个免费的，开源的，简单易用的文件校验器。

## Features 特性
- 支持 Windows 和 macOS 系统
- 支持深色模式
- 校验文件 md5 或 sha256 值
- 生成文件的 md5 或 sha256 值，以便后续使用
