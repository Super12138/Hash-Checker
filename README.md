一个快速、随时可用，且遵循 Material Design 3 的跨平台文件校验器
<br>
<strong>中文（简体）</strong>| [English](README_EN.md)

## ✨ 功能
- 支持 Material Design 3 以及自定义主题色
- 支持 Windows 和 macOS 系统
- 支持深色模式
- 校验和生成文件 MD5 或任何 SHA 值
- 自动检查更新
- PWA（Progressive Web App，渐进式 Web 应用）
